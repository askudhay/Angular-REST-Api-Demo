Angular - Spring REST Demo App

What this App is all about?

A simple Angular app with REST Api as backend developed using Angular, Bootstrap & Spring Boot - REST, that renders perfectly in any size devices / browsers.

Spring Boot - REST

Have been sharing my knowledge on Spring Boot - REST Api development with my fellow associates.

Suddenly one day a thought came to my mind, why not share it with everyone I know in LinkedIn so decided to write an article on "How to develop REST Api using Spring Boot".

Spring Boot LinkedIn Article
Angular 5

To use the REST Api I developed effectively, I was thinking to develop REST Client, then decided to create Angular.

This Single Page Application (SPA) is deployed in AWS Cloud. It interacts with Spring REST Api which is also deployed in AWS Cloud.
